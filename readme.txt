Short description:

This website displays real-time weather information for any city using the OpenWeatherMap API. It features Lottie animations and displays the local time.

Project Overview:

This repository contains the HTML, CSS, and JavaScript code for a weather website.
Users can input a city name to get detailed weather information, including temperature, humidity, wind speed, and more.
The website utilizes the OpenWeatherMap API to fetch weather data.

Technologies Used:

* HTML
* CSS
* JavaScript
* OpenWeatherMap API
* Lottie Animations

Project Structure:

* index.html: Main HTML file for the webpage.
* style.css: CSS file for styling the webpage.
* script.js: JavaScript file for handling user input, API calls, and displaying weather data.
* assets: Folder to store images used in the webpage.
* animations: Folder to store Lottie animation files.

How to Use:

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser.
3. Enter a city name in the search bar and press Search.

Note:

* The website is not responsive and may not display correctly on different screen sizes.

Disclaimer:

This project is for educational purposes only. The accuracy of the weather data is dependent on the OpenWeatherMap API.
